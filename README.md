# informationtheory
assorted information theory scripts

asciilists - from a list of strings of the same length, return the two strings that have the largest and smallest absolute ascii distances

list=["capsule";
"digital";
"perfect";
"dynamic";
"publish";
"private";
"podcast"];

asciilists(list);

podcast
capsule
    97

private
publish
    32
